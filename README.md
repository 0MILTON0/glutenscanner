# GlutenScanner - website

Statisk website for GlutenScanner-appen: en forside med kort beskrivelse af
appen samt appens privatlivspolitik.

## Sider

- `index.html` - forside med beskrivelse af appen
- `privatlivspolitik.html` - privatlivspolitik (bruges bl.a. til app-butikkens listning)
- `404.html` - fejlside (bruges automatisk af GitHub Pages)
- `style.css` - fælles styling for alle sider (understøtter mørk tilstand)

## Lokal visning

Siderne er ren HTML/CSS uden byggetrin. Åbn `index.html` direkte i en browser,
eller start en lille lokal server:

```sh
python3 -m http.server
```
